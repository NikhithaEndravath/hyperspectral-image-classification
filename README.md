# Hyperspectral Image Classification

A machine learning-based project for hyperspectral image classification using semi-supervised learning, superpixel segmentation, and dynamic classifier selection.

## 📌 Overview
This project focuses on improving classification accuracy of hyperspectral images using limited labeled data.

## 🚀 Features
- Semi-supervised learning approach
- Superpixel segmentation
- Dynamic classifier selection
- Improved accuracy
